# 🍽 Food Waste Management System

## 📌 Project Overview
This project is an end-to-end Food Waste Management System designed to efficiently manage, track, and analyze food distribution data. It connects providers and receivers while offering powerful insights through an interactive dashboard.

---

## 🚀 Key Features
- 📦 Manage food listings, providers, and receivers
- 🗄 Structured MySQL database with relationships
- 📊 Interactive dashboard using Streamlit
- 🔍 Advanced filtering and query-based analysis
- 📈 Visual insights (charts, trends, top providers, etc.)

---

## 🛠 Tech Stack
- **Frontend / Dashboard**: Streamlit  
- **Backend**: Python  
- **Database**: MySQL  
- **ORM**: SQLAlchemy  
- **Data Processing**: Pandas  

---

## 📊 Dashboard Highlights
- Power BI-style layout with clean visualization  
- Multiple SQL-based queries (Q1–Q15 insights)  
- Filters for city, provider, and food category  
- Top contributors and distribution trends
